# ResumeFlow

AI-assisted job application toolkit: write your resume once in Markdown, then let Claude Code tailor it to each job description and compile it to PDF.

Built with [Claude Code](https://claude.ai/code) agent skills, Pandoc, and XeLaTeX.

## Quick Start

1. Run **`python3 setup.py`** — enter your name, email, phone, and LinkedIn once
2. Start Claude Code (`claude`)
3. **`/gather-background`** in Claude Code - paste your existing resume or answer questions; Claude builds a structured career profile and optionally drafts your base resumes
4. **`/tailor-resume`** in Claude Code — paste company name, role, and job description; Claude picks the right base resume, rewrites bullets to match ATS keywords, scaffolds a cover letter template for you to write manually, compiles the PDF, and logs the application all automatically.
5. Visit the job directory (`/jobs/NNN_company/resume.md`) and make further edits to your resume as you wish
6. **Write your cover letter** — open `jobs/NNN_company/letter.md` and fill in the body (3-4 paragraphs)
7. **`make open`** - build and open the letter and resume PDFs
8. **`make dashboard`** — generate and open an HTML status board from `tracker.csv`

## Features

- Markdown-first resume authoring — portable, diffable, version-controlled
- Two base resume strategies: technical (projects + engineering skills) vs. general (service + operations)
- AI-assisted ATS keyword targeting via `/tailor-resume` Claude skill
- AI-powered onboarding via `/gather-background` — no blank page
- Manual cover letters — AI scaffolds the header, you write the content
- Application tracking dashboard (HTML, generated from CSV)
- Per-job directories keep every application's source self-contained
- Zero external Python dependencies — just pandoc and xelatex

## Prerequisites

- **macOS or Linux**
- **pandoc** — `brew install pandoc`
- **XeLaTeX** — `brew install --cask mactex` (or install TeX Live on Linux)
- **Python 3.9+** — `python3 --version`
- **Claude Code** — [claude.ai/code](https://claude.ai/code) (for AI tailoring)

## Quick start

```bash
git clone https://github.com/YOUR_USERNAME/resumeflow.git
cd resumeflow

# 1. Enter your contact details
python3 setup.py

# 2. Open Claude Code and build your career profile
#    (in Claude Code terminal)
/gather-background

# 3. Preview your base resume
make build
open output/resume.pdf

# 4. Tailor a resume for a job (in Claude Code)
/tailor-resume
# → paste: Company Name (line 1), Job Title (line 2), full JD (remaining lines)
```

## The Claude Code skills

### `/gather-background`

Run this after `setup.py`. Paste your existing resume or answer questions interactively. Claude writes `markdown/background.md` — a structured profile with your experience, projects, skills, job targets, and writing preferences. Every `/tailor-resume` session reads this file before touching anything.

Optionally drafts `markdown/full_resume.md` and `markdown/general_resume.md` from your answers.

### `/tailor-resume`

The core application skill. Provide:
```
Company Name
Job Title
Full job description text...
```

Claude will:
- Read your background profile
- Choose the right base resume (technical vs. general)
- Scaffold `jobs/NNN_company/` with `resume.md`, `letter.md`, and `Makefile`
- Rewrite resume bullets to surface ATS keywords from the JD
- Fill in the cover letter YAML header (date, company address)
- Compile the resume PDF
- Log the application to `tracker.csv`

You write the cover letter body. Run `make letter -C jobs/NNN_company/` when it's ready.

## Cover letters

ResumeFlow does not auto-generate cover letter content — you write it. The scaffold provides:
- Your contact block (name, email, phone) pre-filled
- Date and company address fields
- A structured comment with suggested paragraph order

Place your handwritten signature image at `template/signature.png` to include it. Remove the `signature:` field from the YAML front matter to omit it.

## Customizing the AI skills

The skills live in `.claude/commands/`. They are plain Markdown files — edit them to change how Claude behaves:

- **`tailor-resume.md`** — controls keyword analysis, editing rules, tracker format
- **`gather-background.md`** — controls the onboarding interview and profile structure
